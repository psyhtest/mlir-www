Detailed Description

Progressive lowering of ContractionOp.

One: x = vector.contract with at least one free/batch dimension is replaced by: a = vector.contract with one less free/batch dimension b = vector.contract with one less free/batch dimension .. x = combine a b .. until a pure contraction is reached (no free/batch dimensions), which is replaced by a dot-product.

This only kicks in when either VectorTransformsOptions is set to Dot or when other contraction patterns fail.
